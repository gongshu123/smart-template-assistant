# 智能套版助手

![应用图标](assets/app_icon.png)

面向 Photoshop 电商页面的本地批量套版工具。使用一个 PSD/PSB 模板、一份标准化 Excel 和一个产品素材文件夹，批量替换商品图片与文字，并输出保留完整图层结构的 PSD 文件。

> 所有文件均在本机处理，不会上传模板、Excel 或商品素材。

## 主要功能

- 自动分析未整理的 PSD/PSB 图层，识别多个商品位。
- 支持产品图、产品名称、产品卖点、价格、利益点、立减和销量。
- 在预览画面中显示变量框，支持点击图层重新绑定。
- 支持 4、6、18 等不同商品位数量的页面模板。
- 自动处理共享智能对象，避免替换一个商品后其他商品同步变化。
- 保留智能对象名称、剪贴关系、混合模式、透明度和图层可见性。
- 价格或立减为 `0`/空时可自动隐藏对应内容。
- 支持先生成第一张测试稿，再批量生成全部 PSD。
- 提供 Excel 数据预览、运行日志、连续缩放和模板映射记忆。

## 运行环境

- Windows 10/11
- Adobe Photoshop 2025 或 2026
- 直接使用 Release 中的 EXE 时无需安装 Python

## 快速使用

1. 从 [Releases](../../releases/latest) 下载最新版 EXE。
2. 打开软件并选择 PSD/PSB 模板。
3. 选择 Excel 数据、产品素材文件夹和 PSD 输出文件夹。
4. 点击“分析模板”，核对每个商品位的变量识别结果。
5. 如识别错误，先选择右侧变量，再点击左侧正确图层重新绑定。
6. 读取 Excel 后先生成第一张测试稿。
7. 检查无误后点击“确认并批量生成PSD”。

## Excel 字段

标准逐行表支持以下字段：

| 字段 | 说明 |
| --- | --- |
| 页面编号 | 同一页面的商品使用相同编号 |
| 商品顺序 | 商品在模板中的位置 |
| 产品图片 | 素材文件夹内的图片文件名 |
| 产品名称 | 页面小标题 |
| 产品卖点 | 页面大标题 |
| 价格 | 到手价或售价 |
| 利益点 | 优惠、赠品等说明 |
| 立减 | 直降金额 |
| 销量 | 已售数量 |

同时兼容“套版01、套版02……”横向套版表。

## 从源码运行

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

## 打包 EXE

```powershell
pyinstaller --noconfirm --clean "智能套版助手.spec"
```

生成文件位于 `dist/智能套版助手.exe`。
